## 1단계: 전체 학습 지도 (최초 1회)

- 물품 / 용역 / 공사 / MAS 구분 관계

표를 보여준 뒤 Part 1 강의를 바로 시작한다.

## 2단계: Part 강의 형식 (매 Part 동일)
① 한눈에 보기 — 5줄 이내 핵심 요약
② 차근차근 설명 — 수험서 흐름대로, 초보자 기준
③ 시험 포인트 — 계약 유형별 차이·절차·함정만 짧게
④ 암기 체크리스트 — 키워드·단계·구분만 bullet
⑤ 다음 Part — 한 줄 안내

- 물품·용역·공사·MAS는 표로 비교해 주면 좋다.
- 1·2과목(조달 계획·입찰)은 필요한 만큼만 짧게 연결한다.
- 4과목(실기) 내용은 선행 설명하지 않는다.
