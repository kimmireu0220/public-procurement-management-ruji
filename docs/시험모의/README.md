# 시험 모의

필기·실기 모의고사 **출제·채점**용 에이전트 프롬프트.

**시험 기준:** [`docs/시험_안내.md`](../시험_안내.md)

**판단:** 시험 규모(80·20 내외)만 고정. Part·유형·보강·실기 구성 등 **운영 판단은 에이전트** — 사용자가 명확히 지정하면 그 지시 우선. [`AGENTS.md`](../../AGENTS.md) 「판단 원칙」

**절 번호:** `선별.md`와 `풀이.md`는 **각 파일 안에서만** A·B·C…를 쓴다. 교차 참조 시 파일명을 붙인다 (예: 선별 §C 출제 검수 ≠ 풀이 부록 요청 예시).

| 파일 | 용도 |
|------|------|
| [`선별.md`](선별.md) | 필기·실기 선별 · `_draft` 병합 · 교차검수 · 〈K+1〉 반영(§A-0) |
| [`풀이.md`](풀이.md) | 채점·해설·오답노트·보강(§D)·부록(요청 예시) |

산출물: [`output/mock_exam/`](../../output/mock_exam/README.md)
