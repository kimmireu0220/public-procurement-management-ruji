# 형식 일괄 수정 프롬프트 (전 과목)

기존 `output/agent_extract/<slug>/partN.md`의 **형식만** 고칠 때 사용. 문항 내용·번호·정답은 바꾸지 않는다.

## 스크립트 (권장)

| 도구 | 용도 |
|------|------|
| `python3 tools/fix_choice_lines.py --subject N` | `①②③④` 한 줄 몰아쓰기 → 줄 분리 |
| `python3 tools/annotate_source_ranges.py --part6` | 1과목 Part 06 출처 `(문항 N~M)` 범위 표기 |

수정 후 `python3 tools/build_problem_book.py --subject N` 실행.

## 에이전트 수동 프롬프트

```
너는 `output/agent_extract/〈slug〉/part〈N〉.md`의 **형식만** 수정하는 편집자다.
문항 내용·번호·정답은 바꾸지 않는다.

## 작업
1. 객관식: `N. 문제? ① … ② … ③ … ④ …` 한 줄 형태를 찾아
   - `N. 문제?` 한 줄
   - `   ① …` / `   ② …` / `   ③ …` / `   ④ …` 각각 한 줄로 분리 (3칸 들여쓰기)
2. 수정 후 `python3 tools/build_problem_book.py --subject 〈과목번호〉` 실행

## 금지
- 정답 섹션 수정
- 문항 삭제·추가
- 암기팁 일괄 추가 (학습 시 필요하면 그때 에이전트에게 요청)

## 자가검사
- 문항 본문 줄에 `①.*②` 패턴이 남아 있으면 선지 분리 미완료
```
