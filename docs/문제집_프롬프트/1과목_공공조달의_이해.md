## A. Part 추출 (Part 〈N〉 작업 시 〈N〉·〈총페이지수〉만 교체)

```
너는 공공조달관리사 박문각 수험서 "1과목_공공조달의 이해"의 Part 〈N〉에서 **문제만** 추출하는 작업자다.

## 입력 자료
- 원본(정확도 기준): `$PROJECT_ROOT/sources/민간_박문각_수험서_jpg/1과목_공공조달의 이해/Part 〈N〉/page_0001.jpg` ~ `page_〈총페이지수, 4자리〉.jpg`
- OCR 보조: `$PROJECT_ROOT/output/ocr/1과목_공공조달의_이해/Part 〈N〉/page_XXXX.txt`
- JPG를 Read(vision)로 직접 판독. CHAPTER는 Part 안 소단원.

## 작업 절차
1. Part 〈N〉 페이지를 순서대로 훑어 CHAPTER·문제 섹션 구조 파악 (Check Q&A, 단원별 출제예상문제, 최종점검 OX).
2. 문제 페이지 JPG 복원. 작은 글씨는 ImageMagick `convert` 크롭·2~3배 확대 가능.

## 포함 / 제외
- 포함: Check Q&A, 단원별 출제예상문제, 최종점검 OX
- 제외: 이론 본문, 표지·목차, 손글씨 풀이 흔적, 해설·정답 본문

## 출력 형식 (엄수)
- 객관식: `N. 문제문` 한 줄 + `①`~`④` **각각 한 줄** (3칸 들여쓰기)
- **금지**: `N. 문제? ① … ② … ③ … ④ …` 한 줄에 몰아쓰기
- OX: `N. 문제문 (O/X)`
- 암기팁 작성하지 않음
- 문항마다 `<!-- source: Part 〈N〉/page_XXXX.jpg -->`
- 정답: 파일 끝 `## Part 〈N〉 정답 및 해설` 섹션에만
- 출력: `$PROJECT_ROOT/output/agent_extract/1과목_공공조달의 이해/part〈N〉.md`

## 저장 전 자가검사
- 문항 본문 줄에 `①.*②`가 있으면 선지 분리 후 저장

## 완료 보고
섹션별 문항 수, 페이지 범위, 애매·오타 의심 건 요약.
```

## B. 독립 검수

```
너는 1과목 문제집 **독립 품질 검수자**다. `output/agent_extract/1과목_공공조달의 이해/partN.md` (part1~7)를 검수·수정하라.
원본: `sources/민간_박문각_수험서_jpg/1과목_공공조달의 이해/Part N/page_XXXX.jpg`

## 검수 항목
1. 객관식 선지 줄바꿈 (문항 줄에 ①②③④ 붙어 있으면 분리)
2. 형식 일관성 (번호, OX, 정답부)
3. 본문 정답·해설 혼입 여부
4. 문항 수 = 정답 수
5. 깨진 문장·이상 문자
6. Part별 표본 1~2페이지 JPG 대조

```
