# 문제집 프롬프트

과목별 **문제 추출·검수** playbook. Part 폴더 작업 시 해당 과목 파일을 채팅에 붙여 넣는다.

> **현재 상태 (2026-06):** 전 과목(1~4) 23개 Part 추출·빌드·독립 검수 완료. 아래 파일은 **재추출·재검수·형식 수정** 시 사용.

## 왜 과목별 파일인가

초기에는 전 과목 공통 템플릿 하나(`문제집_제작_프롬프트.md`)에 `〈slug〉`·`〈N〉`만 바꿔 쓰는 방식이었다. Part당 에이전트 1개로 일괄 추출할 때는 편했지만, 과목마다 Part 수·페이지 수·품질 이슈가 달라 **`학습_프롬프트/`와 같이 과목별 파일**이 더 낫다.

## 공통 규칙

- 추출 형식(선지 줄바꿈): [`extraction_guide.md`](../extraction_guide.md)
- **시험 일정·문항 수:** [`시험_안내.md`](../시험_안내.md)
- 형식만 일괄 수정: [`형식_수정.md`](형식_수정.md)
- 시험 규모 모의 선별: [`시험모의_선별.md`](시험모의_선별.md)
- 모의 풀이·해설·오답노트: [`시험모의_풀이.md`](시험모의_풀이.md)

## 용어

- 경로·source: `sources/민간_박문각_수험서_jpg/<slug>/Part N/page_XXXX.jpg`
- 1과목 `Intro` = 출제기준·목차 (문제 추출 제외)
- 교재 내부 `CHAPTER 01` 등 = Part 안 소단원
