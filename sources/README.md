# sources — 외부 학습 자료

조달관리사 시험 준비에 쓰는 **외부 원본 자료**만 둔다. `tools/` 파이프라인 산출물은 `output/`에 있다.

## 출처·용도

| 경로 | 발행·배포 | 공식성 | 형식 | 프로젝트 용도 |
|------|-----------|--------|------|----------------|
| [`민간_박문각_수험서_jpg/`](민간_박문각_수험서_jpg/) | ㈜박문각 | **민간** 수험서 | JPG 스캔 | OCR · agent_extract · 문제집 **주 입력** |
| [`공식_조달청_표준교재_pdf/`](공식_조달청_표준교재_pdf/) | 조달청 발간 · [공공조달역량개발원](https://hrd.pps.go.kr) 배포 | **공식** 표준교재 | PDF | 내용 대조 · 참고 |
| [`공식_qnet_예제문제/`](공식_qnet_예제문제/) | [한국산업인력공단 Q-Net](https://www.q-net.or.kr) 공지 첨부 | **공식** 예제 | PDF | 필기 출제 방향 참고 (6문항, 정답·해설 없음, 기출 아님) |
| [`공식_qnet_시행공고/`](공식_qnet_시행공고/) | Q-Net 수시검정 시행공고 | **공식** 공고 | PDF | **일정·CBT/필답·접수 시각** (문항 수는 출제기준·`docs/시험_안내.md`) |

박문각 수험서는 **조달청 표준교재가 아닙니다.** 표준교재·출제기준을 참고해 집필된 상업 수험서이며, 본 저장소의 문제집 추출은 이 스캔을 기준으로 한다.

## 과목 하위 폴더 (4권 공통)

| 과목 | slug |
|------|------|
| 필기 1 | `1과목_공공조달의 이해` |
| 필기 2 | `2과목_공공조달 계획분석` |
| 필기 3 | `3과목_공공계약관리` |
| 실기 4 | `4과목_공공조달 관리실무` |

스캔·PDF 모두 위 slug로 Part 폴더를 맞춘다. 1과목 스캔만 `Intro`(출제기준·목차)가 있다.

## Cursor 첨부 예시

```
@sources/민간_박문각_수험서_jpg/1과목_공공조달의 이해
@output/ocr/1과목_공공조달의_이해
```
