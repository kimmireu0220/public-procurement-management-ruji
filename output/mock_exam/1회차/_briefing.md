## 에이전트 워크플로 (K회차)

1. **과목별 선별** — `_candidates/{N}과목_part*.md` 에서만 고른다.
   - 산출: `_draft/{N}과목_선별.md` (문항 전문 + `<!-- id: ... -->` 유지)
   - 또는 `_draft/manifest.json` 에 id 목록만 기록
2. **교차 검수** — `docs/문제집_프롬프트/시험모의_선별.md` A-3 프롬프트
3. **병합** — `python3 tools/merge_mock_draft.py 1` (HTML + 브라우저 자동 열기)
4. **검증** — `python3 tools/validate_mock_exam.py 1`

급할 때 초안: `python3 tools/select_mock_exam.py 1 --importance` 후 에이전트가 10~15문항만 교체.
